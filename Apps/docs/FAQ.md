|应用是否将任何 Microsoft 客户数据或客户内容传输到第三方或子处理器？|客户数据可以是员工用户名或 ID、位置、人员信息、用户特定 IP 地址等。如果组织将任何 Microsoft 客户数据或客户内容传输到第三方或子处理器，请选择“是”。 如果没有，请选择“否”。|
|是否与共享 Microsoft 客户数据的任何第三方服务达成数据共享协议？|第三方服务可以是呼叫中心、BPO、数据输入等。如果同意与这些第三方服务共享 Microsoft 客户数据，请选择“是”。 如果没有，请选择“否”。|
